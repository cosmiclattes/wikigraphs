Editor Cohort Activity - Value

What is this graph?

This graph shows the total edit activity by an editor cohort over its lifetime.

How do I read the graph?

  • When an editor has edits >= 5/month the editor is considered active.
  • Editors are grouped by the month of their first edit - editor cohort.
  • X-axis (time since the start of the wiki)
  • Y-axis (editor cohorts)
  • Each row gives the total edit sessions for a given editor cohort.
  • Each data point gives the edit sessions for a given editor cohort in a given month since birth.
  • The brush lets you look at the monthly activity of the cohorts. For ex, to see the activity of every cohort in its first month select 1-2 in the filter.
