Editor Cohort Bytes Added - Value

What is this graph?

This graph shows the total bytes added by an editor cohort over its lifetime.

How do I read the graph?

  • X-axis (time since the start of the wiki in months)
  • Y-axis, Editors are grouped by the month of their first edit - editor cohort.
  • Each data point gives the number of bytes added to an editor cohort in a month since creation.
  • The brush lets you filter by the month of activity in an editor cohort.
