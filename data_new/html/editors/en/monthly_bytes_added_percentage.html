Monthly Bytes Added - Percentage

What is this graph?

This graph shows the total bytes added in a month split by the contribution in percentage from each editor cohort.

How do I read the graph?

  • X-axis (time since the start of the wiki in months)
  • Y-axis (editors are grouped by the month in which they were created - editor cohorts)
  • Each data point gives the percentage of bytes added by an editor cohort in a month.
  • The denominator is the total number of bytes added in a month.
  • The brush lets you filter by the age of an editor cohort.
