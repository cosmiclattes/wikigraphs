Editor Cohort Retention Rate

What is this graph?

It shows how the edit activity on of editors who joined in a given month changes month over month.

How do I read the graph?

  • When an editor has edits >= 5/month the editor is counted as active. If editor has only a single edit it isn't counted.
  • X-axis (time since the start of the wiki in months)
  • editors are grouped by the month where they made their first edit - editor Cohorts
  • Y-axis (editor cohorts)
  • Each row in the graph represents the activity of an editor group, eg: Editors who made their first edit in Jan 05.
  • Each data point in a row gives the percentage change of editors who were active in the current month over the previous month in that given row (editor cohort).
  • The brush lets you filter by the percentage values of the data points.
